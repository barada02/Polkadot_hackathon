PolkaFlow is an intelligent multi-chain portfolio and fee optimization platform for the Polkadot ecosystem. It provides real-time portfolio analysis, cross-chain fee comparison, and optimal route finding across multiple parachains.

### Key Features
- **Multi-Chain Portfolio Analysis** - Real-time portfolio tracking across 6 Westend parachains
- **Fee Optimization Engine** - Compare fees and find cheapest routes (up to 89% savings)
- **Network Health Monitoring** - Real-time status of all supported chains
- **Interactive Dashboard** - Clean, responsive UI for seamless user experience

### Live Demo
- **Frontend**: [https://polkaflow-five.vercel.app](https://polkaflow-five.vercel.app)
- **Backend API**: [https://polkaflow-backend.onrender.com](https://polkaflow-backend.onrender.com)

### Technology Stack
- **Backend**: Node.js, Express, Polkadot API v1.20.6
- **Frontend**: React 19, TypeScript, Vite, Chart.js
- **Deployment**: Render (Backend), Vercel (Frontend)
- **Supported Networks**: Westend Ecosystem (6 parachains)
